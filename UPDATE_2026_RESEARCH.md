## 1. Складываются ли кэшбэк + скидка на сайте + реферал + промокод + скидка по банковской карте?

### ✅ ДА — Полное комбинирование возможно

Согласно исследованию 2025-2026 годов, **максимальный стеккинг** включает следующие слои:

```
Формула максимальной экономии:
┌─────────────────────────────────────────────────────────┐
│  1. Скидка на сайте (промокод/акция) ......... до 30%  │
│  2. Кэшбэк-портал ................................ 6%  │
│  3. Реферальная программа ..................... до 10%  │
│  4. Промокод (первый заказ/подписка) .......... до 20%  │
│  5. Скидка по банковской карте ................ до 12%  │
│  6. Карта лояльности магазина ................. до 8%   │
│  7. CreamSale Coins ........................... до 5%   │
│  ─────────────────────────────────────────────────────  │
│  ИТОГО МАКСИМАЛЬНАЯ ЭКОНОМИЯ: ............... до 75%+  │
└─────────────────────────────────────────────────────────┘
```

### Разрешённые комбинации (2026):

| Слой | Тип | Можно комбинировать |
|------|-----|---------------------|
| **Скидка на сайте** | Автоматическая/промокод | ✅ Да |
| **Кэшбэк-портал** | Один портал за раз | ✅ Да |
| **Реферальная программа** | Код друга | ✅ Да |
| **Промокод** | Один за транзакцию | ✅ Да |
| **Банковская карта** | Кэшбэк категории | ✅ Да |
| **Карта лояльности** | Баллы магазина | ✅ Да |
| **CreamSale Coins** | Внутренняя валюта | ✅ Да |

### ⛔ Ограничения:

1. **Только 1 кэшбэк-портал** — нельзя использовать несколько порталов одновременно
2. **Только 1 промокод** — если магазин разрешает один код
3. **Исключительные коды** — пометка «exclusive» может не сочетаться с кэшбэком
4. **Соблюдение условий магазина** — необходимо читать правила ретейлера

### 📋 Пример реального стекинга:

```
Покупка: Apple AirPods Pro ($249.99)

Применяется:
├─ Скидка на сайте (Black Friday) ......... -$50.00 (20%)
├─ Кэшбэк-портал (Goodshop) ............... -$12.00 (6%)
├─ Реферальный код друга .................. -$15.00 (6%)
├─ Промокод для новых пользователей ....... -$20.00 (8%)
├─ Кэшбэк банковской карты (5% категории) . -$7.65 (5%)
├─ Карта лояльности (баллы) ............... -$10.00
└─ CreamSale Coins (накопленные) .......... -$12.35 (5%)

ИТОГО: $249.99 → $122.99
ЭКОНОМИЯ: $127.00 (51%)
```

### 🛡️ Анти-скрапинг меры и обход (2026)

**Проблемы:**
- CAPTCHA и Cloudflare
- Блокировка по IP
- Динамическое ценообразование
- Геоблокировка

**Легальные подходы:**

| Метод | Описание | Законность |
|-------|----------|------------|
| **Официальные API** | Прямое подключение к API маркетплейсов | ✅ Полностью |
| **Партнёрские сети** | CJ, Rakuten, Impact (агрегируют данные) | ✅ Полностью |
| **Прокси-ротация** | Гео-распределённые запросы | ⚠️ Серая зона |
| **Браузерная автоматизация** | Puppeteer, Playwright | ⚠️ Серая зона |
| **Пользовательские данные** | Краудсорсинг цен от пользователей | ✅ Полностью |

## 5. Юридические аспекты

### ✅ Разрешено:
- Использование официальных API
- Партнёрские программы (affiliate)
- Пользовательский контент (скриншоты, чеки)
- Браузерное расширение для личного использования

### ⚠️ Серая зона:
- Автоматизированный парсинг без API
- Обход CAPTCHA
- Создание фейковых аккаунтов для сбора данных

### ❌ Запрещено:
- Нарушение условий использования API
- Коммерческое использование украденных данных
- Вмешательство в работу сайтов (DDoS-подобные запросы)

### 📋 Рекомендации:
1. Использовать только официальные API где возможно
2. Реализовать краудсорсинг как основной источник данных
3. Партнёрства с affiliate-сетями (CJ, Rakuten, Impact)
4. Прозрачная политика конфиденциальности
5. Compliance с GDPR/CCPA для пользователей из ЕС/Калифорнии

## 6. Выводы и рекомендации

### 🎯 CreamSale жизнеспособен в 2026 потому что:

1. **Рынок растёт** — $15.8 млрд → $28.4 млрд к 2030
2. **Конкуренты фрагментированы** — нет единого решения для всего
3. **Пользователи хотят автоматизации** — устали от 5-7 приложений
4. **Технологии позволяют** — API, AI, краудсорсинг

### 🔑 Ключевые факторы успеха:

| Фактор | Приоритет | Действия |

| **Интеграции API** | Высокий | Amazon, eBay, Walmart, affiliate сети |
